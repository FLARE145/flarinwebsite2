Digital

This one is somewhat confusing to understand, so I feel like I should explain a bit. This was done for Pico Day 2021 on Newgrounds. It's inspired by the game Pico's School, but deviates greatly from what that game actually is. Tom Fulp's original game simply exists to be played. It's not trying to make some kind of social comment like this artwork might imply. School shootings are a heavy topic, and I wanted to portay a character that feels overwhelmed by that.
